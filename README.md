## 🇺🇦 Українська

**PyInstarCreate** — це графічний інтерфейс для генерації команд `PyInstaller`. Він створений, щоб спростити життя Python-розробникам, позбавляючи потреби вводити довгі та складні команди в терміналі вручну.

### ✨ Особливості:
* **Швидке налаштування:** Легко вибирайте між `--onefile` (один файл) та `--onedir` (одна папка).
* **Управління вікнами:** Перемикайте режими консолі (`--console` / `--noconsole`) одним кліком.
* **Підтримка ресурсів:** Зручне додавання іконок (`.ico`) та додаткових файлів.
* **Копіювання в один клік:** Генеруйте команду та миттєво копіюйте її для використання.

### 🚀 Як почати:
1. Клонуйте репозиторій: `git clone https://github.com/yourusername/PyInstarCreate.git`
2. Встановіть залежності: `pip install -r requirements.txt`
3. Запустіть програму: `python main.py`

<a name="english"></a>
## 🇺🇸 English

**PyInstarCreate** is a GUI-based command generator for `PyInstaller`. It is designed to streamline the workflow for Python developers by eliminating the need to manually type long and complex commands in the terminal.

### ✨ Features:
* **Quick Configuration:** Easily switch between `--onefile` and `--onedir` modes.
* **Window Management:** Toggle between console and windowed modes (`--console` / `--noconsole`) with a single click.
* **Resource Support:** Simple integration of icons (`.ico`) and additional data files.
* **One-Click Export:** Generate your command and copy it to the clipboard instantly.

### 🚀 Getting Started:
1. Clone the repository: `git clone https://github.com/yourusername/PyInstarCreate.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python main.py`
